WordPress Authority Checker plugin: This plugin will allow you to add authority checker tools directly to your website with no limit of usage.

== Description ==

Domain Authority, Page Authority and moz rank are those things which every webmaster uses to check quality of their website or webpage. 
This plugin will allow you to add [website authority checker](https://www.prepostseo.com/domain-authority-checker), domain authority, page authority and moz rank checker tools to your webpage with just a single short code.
This plugin will send requests to [moz.org](https://moz.com/) to calculate domain and page authority of any webapge.

= What is Domain Authority =
Domain authority of a website can be 1-100 and it’s calculated and updated by moz on monthly basis. Moz check hundreds of factors to calculate DA of a website. It checks total number of links pointed to your root domain, quality and speed of your website, Age of your domain, SSL certificate and moz trust. [Domain Authority Checker](https://www.prepostseo.com/domain-authority-checker) tool is the need of every website owner. You can also check details about domain authority in the video below.

= What is Page Authority =
Page Authority of is different for each page of your website. Every new page starts is PA from 0 and it is on the number of backlinks, how much you can make it higher. Page authority of a url can be 1-100. Online [Page Authority Checker](https://www.prepostseo.com/page-authority-checker) is mostly used to check the rating of the current webpage instead of the whole website.

= What is Moz Rank =
Moz Rank and Moz trust are those term which were introduced by moz.org after google stopped updating page rank for the websites. Moz use these factors to calculate DA / PA of a webpage. 
Term links equity is also used to calculate the website authority, it means the total number of links to that specific webpage.

= Is this free Tool =
This plugin is 100% free can you can use it on any post or page but just adding a single short code. Which is [authority-checker-tool].



== Installation ==

You can use the built in installer and upgrader, or you can install the plugin
manually.

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit your Options Page to check plugin status.
1. Configure any options as desired, and then enable the plugin
1. That's it!
